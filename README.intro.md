# Introduction & Overview

## Project Vision

The LEGO Sample Factory Control System is a modern, microservice-based Manufacturing Execution System (MES) designed for LEGO production, inventory, and order management. It demonstrates best practices in cloud-native architecture, rapid prototyping, and real-time operations for manufacturing environments.

## Features at a Glance
- Role-based dashboards (Admin, Warehouse, Modules, Production)
- Real-time inventory and order management
- Production scheduling with SimAL integration
- JWT authentication and secure API gateway
- Responsive UI with live updates and notifications

## User Roles
- **Admin**: System KPIs, user management, configuration
- **Warehouse**: Customer order fulfillment, stock management
- **Modules Supermarket**: Module warehouse operations
- **Production Control**: Production order management, scheduling

## UI/UX Highlights
- Responsive design for desktop/tablet
- Statistics cards and color-coded status badges
- Standardized table styling and compact layouts
- Notification system for feedback
